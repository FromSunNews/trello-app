# ReactJS + Hook + Redux - Building an app like Trello from scratch.

## Requirements

* **nodejs >= v16.17.0
* **npm >= 8.15.0

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn install
$ yarn start
```
